# Scientific Scope

OmicsTrust is a trust and failure-analysis platform for omics workflows. CellAudit evaluates single-cell matrices using data QC, spectral structural inference, empirical-null validation, confounding risk, stability, baseline comparison, and reproducibility capture.

The platform may say that a component is statistically detectable above an empirical null. It must not say that the component is definitely a biological discovery.

CellAudit is appropriate for reviewing analysis credibility, identifying technical risks, and guiding follow-up validation. It is not a universal method-ranking system, an automatic biological discovery engine, or a substitute for experimental design review.
